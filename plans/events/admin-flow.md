# Admin Flow

This is a simple admin flow for the event system.

## 1. Create Event

- Choose a fund cycle.
- Create a new event under that cycle.
- Add title, description, status, and dates.

## 2. Add Packages

- Create one or more packages for the event.
- Set package name, price, and advance payment percent.
- Set minimum and maximum quantity if needed.
- Set stock quantity.

## 3. Add Pickup Points

- Create multiple hubs or pickup points.
- Add name, area, address, contact person, and phone.

## 4. Publish Event

- Mark the event as public.
- Open order window.
- Keep the event visible on the public site.

## 5. Monitor Orders

- View incoming orders.
- Check payment status.
- Confirm or cancel orders when needed.

## 6. Track Event Progress

- Update order status.
- Monitor delivery or pickup progress.
- Keep notes and logs simple.

## Minimal Principle

- Keep the admin flow short.
- Use clear status fields.
- Avoid complex workflows in the first version.
