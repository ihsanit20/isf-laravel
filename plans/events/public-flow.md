# Public Flow

This is a simple public flow for event-based pre-orders.

## 1. View Events

- Show published events on the public site.
- Display event title, image, description, and delivery date.

## 2. Choose Package

- Open one event.
- Select a package.
- Select quantity.
- Show price and advance payment amount.

## 3. Select Pickup Point

- Choose a hub or pickup point.
- Show address and contact info.

## 4. Place Order

- Enter name, phone, and delivery details.
- Submit the order.
- Keep the order pending until advance payment is completed.

## 5. Confirm Payment

- After payment is received or verified, confirm the order.
- Generate an order number.
- Send the order number by SMS.

## 6. Track Order

- Allow the user to track using order number and phone number.
- Show order status clearly.

## Minimal Principle

- Keep the public flow short and easy.
- Focus on pre-order, payment, confirmation, and tracking.
- Avoid extra steps in the first version.
